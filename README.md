# Site para fisioterapeuta

Site profissional para fisioterapeuta com apresentacao, tratamentos, depoimentos e formulario de agendamento.

## Arquivos

- `index.html`: estrutura da pagina
- `styles.css`: estilos responsivos
- `script.js`: envio simulado do formulario de agendamento

## Publicacao

Pode ser hospedado no GitHub Pages usando a branch `main` e a pasta raiz do repositorio.
